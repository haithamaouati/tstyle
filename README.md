# tstyle

**tstyle** is a small, no-nonsense script for Termux that lets you quickly change color schemes and fonts without digging through configs or memorizing escape codes.
Minimal. Fast. Made for people who live in the terminal.

![tstyle preview](preview.png)

## Features

- Change Termux color schemes instantly
- Switch between different fonts
- Lightweight Bash script
- No heavy dependencies
- Designed specifically for [Termux](https://termux.dev/en/) on Android

## Installation

1. Clone the repository
```bash
git clone https://github.com/haithamaouati/tstyle.git
```
2. Change to the tstyle directory:
```
cd tstyle
```
3. Install the script
```
./install.sh
```

## Usage

Usage: `tstyle`

##### Options:

1. Colors (523)
2. Fonts (21)
3. Random
0. Exit

## Uninstallation

```
./uninstall.sh
```

## Environment

- Tested on [Termux](https://termux.dev/en/) only!
